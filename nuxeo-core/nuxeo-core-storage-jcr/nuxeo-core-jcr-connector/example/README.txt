An example of how to use NXCore.
You can build your own application on top of NXCoreApplication (as in the example provided by NXCoreAppExample)
To run the application you need to add on yout classpath the libraries:
NXRuntime.jar
NXCore.jar
NXCoreAPI.jar
NXJCRConnector.jar
NXSchema.jar
NXQuery.jar

and all the external dependences:
log4j.jar
commons-logging.jar
java-cup-v11a.jar
JFlex.jar ??
jcr.jar
jackrabbit-core-1.1.jar

and all jackrabbit dependencies
concurrent-1.3.4.jar
commons-collections-3.1.jar
xercesImpl-2.6.2.jar
slf4j-log4j12.jar
