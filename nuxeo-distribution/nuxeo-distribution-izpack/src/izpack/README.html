Note: Nuxeo EP needs Java 1.5

It won't currently work with either a previous (e.g. Java 1.4) or later (e.g Java 6 or 1.6) release of Java. Please set up JRE 1.5 if it's not already installed on your computer.

Getting Started

After installing this application using this installer:

  1. Go to your installation directory
  2. Start the application using the appropriate "run" command in the "bin" directory (ex: "sh ./bin/run.sh" under Unix/Linux/MacOS, or "bin\run.bat" under Windows)
  3. Point your browser to http://localhost:8080/nuxeo/
  4. Log in using the "Administrator" login and the "Administrator" password (no quotes, of course)

Note: If you want to get the PDF transformation works, and if you want the Microsoft Office documents to be indexed, you need to complete the installation by following the instructions here.

About Nuxeo 5

Nuxeo 5 is a robust, extensible, global Enterprise Content Management (ECM) solution available as Open Source Software (OSS). Nuxeo imagined, developed and is releasing the global solution Nuxeo 5, helped by a vibrant community of professional and individual contributors.

Nuxeo 5 is designed to enable building and running a large range of applications within the ECM scope, closely matching enterprises needs from the server to the desktop. Nuxeo 5 is a global solution ready to take on challenges of this new era of information management (Web 2.0, increasing quantity of digital information, lots of emails, online/offline, mobility, etc.).

Nuxeo 5.1 main features:

  • Collaborative workspaces
  • Access rights management
  • Theme manager / Ajax support
  • Document type & metadata schemas
  • Audit / History status
  • Directories
  • Business Process Management / Workflow
  • Relation engine
  • Search engine
  • Desktop integration using Firefox, IE & Microsoft Office plugins
  • RSS feature
  • Forum

This release presents the main features of the platform, through a default web client.
