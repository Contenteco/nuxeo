Template displaying additional content for an action, typically a fancy box content. Since 5.7, also generates a javascript function to handle confimation message. Parameters: - action: the action instance (mandatory) - actionWidgetBinding: binding actions displaying widgets should be bound to (mandatory if widgets need it) - idPrefix: a String prefix to put before the action id (optional but recommanded) - actionType (optional): type to use if not set on action or needs to be used by default if not set on action
