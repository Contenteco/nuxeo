Template displaying an action button (@since 5.6). Parameters: - action: the action instance (mandatory) - idPrefix: a String prefix to put before the action id (optional but recommanded) - useAjaxForm: boolean expression whether the action is done in an ajax form: the action will be executed via an action listener to prevent any navigation that would remove the ajax behaviour, defaults to false. - display: if set to 'buttons', action will be displayed as a command button. If set to 'links', action will be displayed as a command link. If set to 'icons', action will be displayed as an icon. Else (default behaviour), action will be displayed as command link and display its icon when defined. Templating zone that can be filled when useAjaxForm is set to false (default): - inside_action Templating zones that can be filled when useAjaxForm is set to true: - before_action_trigger - after_action_trigger
